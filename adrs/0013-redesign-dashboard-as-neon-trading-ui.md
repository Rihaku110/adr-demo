# 0013. ダッシュボードをトレーディング風ネオン UI にリデザイン

- ステータス: 採用
- 日付: 2026-05-08

## 決定

`Kyndryl Dashboard` の見た目を、トレーディング／金融プロダクト風の「ネオン × グラスモーフィズム」スタイルに全面リデザインする。主な変更は以下のとおり。

- 既定テーマを `dark` に変更し、深夜トーン（`#0a0e1f` 基調）＋シアン/バイオレット/ピンクの三色グロー光源と、薄いグリッドノイズ背景を追加
- ナビバー: ブランドマークを四角アイコン化、`Kyndryl Dashboard` を主従構造に、`LIVE` ステータスピル（パルスアニメーション付き）と簡素なアイコン式テーマトグルを設置
- ヒーロー: `eyebrow` を `NYSE: KD` ティッカーバッジ + ラベルの 2 段構成に、タイトルにシアン→バイオレット→ピンクのグラデーションと弱いネオングロー
- KPI カード: 「最新値（前期比 ▲▼ 数値+%）」「平均」「ピーク」「期間変化」の 4 枚に再構成。値はモノスペース、上昇は緑・下落は赤、上端 2px のレインボーアクセントライン
- チャート: 棒は `cyan→violet`、折れ線は `cyan→violet→pink` のグラデーションストロークに `feGaussianBlur` ベースの `glow` フィルタを適用。背景に微細グリッドを追加し、ツールチップはダーク + シアン枠
- 共通: モノスペース指定 (`var(--mono)`)、`backdrop-filter` 強化、CSS 変数 (`--accent-*`, `--up`, `--down`) で配色を集中管理

## 根拠

- 「もっとカッコよく」という要件に最もフィットするのは、データ系プロダクトでよく見るブルームバーグ／TradingView 風の暗色ネオン UI。Kyndryl の銘柄を扱うアプリ性質とも整合する。
- ダーク基調にすることでチャートのグロー効果が活きる。SVG の `feGaussianBlur` + `feMerge` による発光は CSS では再現が難しく、ライブラリ非依存で「らしさ」を一気に出せる手段として最適。
- KPI を「最新値 + デルタ」中心に組み替えたのは、株価／業績ダッシュボードでユーザーが最初に求める情報が「直近どう動いたか」だから。`▲ +2.8 (+6.6%)` のような表示は金融 UI の慣習に合致する。
- 既存のアクセシビリティ（`aria-label`, `role`, `aria-live`）と DOM 構造はそのまま温存。CSS と一部の JSX 構造変更のみで済むため、データレイヤや状態管理には影響しない。

## 代替案

1. **配色だけ変える（ライト基調のまま）**: 差分は最小だが、ネオングロー・グリッド背景といった「カッコよさ」の主成分が活きない。要件の趣旨に合わず不採用。
2. **Tailwind / shadcn 等の UI フレームワークを導入**: 一定の見栄えは保証されるが、依存追加と既存スタイルの整理コストが大きい。プロジェクト規模に対し過剰。
3. **3D / Three.js による派手な背景**: 視覚的インパクトは大きいが、可読性とパフォーマンスが落ちダッシュボードの主役（数値）が霞む。SVG グロー＋CSS グラデーションに留め、主役を数値とチャートに保った。
4. **チャート背景の罫線を実線で密に描く**: トレーディング感は強くなるが、サンプル数 12 程度のデータでは情報過多になる。`stroke-dasharray` + `mask-image` で控えめに表現し、数値の視認性を優先した。
