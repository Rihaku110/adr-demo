# 0020 main.jsx に hello ログを追加

## 決定 (Decision)
`react-app/src/main.jsx` の末尾に `console.log('hello')` を追記する。

## 根拠 (Rationale)
既存の起動ログ（`app booted` / `ready`）に続いて、追加の動作確認用ログを出力することで、ADR ゲート運用とログ出力フローが期待通り機能していることを目視で確認しやすくする。アプリのロジックや UI への影響はなく、リスクが極めて低い変更で挙動確認を行える。

## 代替案 (Alternatives)
- **既存ログを書き換える**: 既存ログ（`ready` 等）の意味が変わってしまい、過去 ADR との整合性を損なうため不採用。
- **ログを追加しない**: 動作確認の手掛かりが増えず、軽微な変更を通じた運用検証ができないため不採用。
- **`console.info` などレベル違いを使う**: 既存の `console.log` 群と統一されず一貫性が崩れるため不採用。
