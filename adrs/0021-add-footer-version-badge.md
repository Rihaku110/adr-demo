# 0021 フッターにバージョン表示を追加

## 決定 (Decision)
`react-app/src/App.jsx` の `Footer` コンポーネント内に、`footer-version` クラスを付与した `<span>` を追加し、コピーライト行末尾に ` · v1.0` を表示する。アクセシビリティ用に `aria-label="version"` を付与する。

## 根拠 (Rationale)
- ユーザーから「小さな UI 変更」の依頼があったため、レイアウト崩れを起こさずに追加できる軽微な情報表示（バージョン）を選択した。
- フッターは情報密度が低く、視認性に影響を与えずに追加情報を入れるのに適している。
- `aria-label` を付与することでスクリーンリーダー利用者にも意味を伝達でき、アクセシビリティを損なわない。
- 既存の `footer` クラスと並列に追加するだけなので、CSS 未定義でも壊れず、必要に応じて後からスタイル付与可能。

## 代替案 (Alternatives)
- **ヘッダー (`Hero`) に変更を加える案**: ヒーローは視認性が高くレイアウト調整の影響が大きいため、「小さな UI 変更」の趣旨から外れると判断し不採用。
- **`Navbar` の `LIVE` バッジ近くにバージョンを表示する案**: ナビゲーション領域は要素が密集しており、追加するとレイアウト調整が必要になるため不採用。
- **コミットハッシュ等を動的取得して表示する案**: ビルド設定変更が必要で「小さな UI 変更」ではないため不採用。
